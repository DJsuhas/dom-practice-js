# DOM Practice – JavaScript (Part 9)

This project contains basic DOM manipulation practice questions solved using pure JavaScript.

All elements are dynamically created using JavaScript only (no pre-written HTML elements).

---

##  Questions Covered

###  Q1
Create a new `input` and `button` element using JavaScript only.  
Set the button text to **"Click me"**.

###  Q2
Add the following attributes:
- Change placeholder of input to **"username"**
- Change the id of button to **"btn"**

###  Q3
Access the button using `querySelector` and change:
- Background color → **blue**
- Text color → **white**

###  Q4
Create an `h1` element:
- Text → **DOM Practice**
- Underlined
- Color → **purple**

###  Q5
Create a `p` tag:
- Text → `Apna College Delta Practice`
- Make **Delta** bold

##  How to Run

1. Download or clone the project.
2. Open `index.html` in your browser.
3. All elements will be generated dynamically using JavaScript.

---

##  Expected Output

The page will display:

- An input field with placeholder **username**
- A blue button with white text saying **Click me**
- A purple underlined heading **DOM Practice**
- A paragraph: `Apna College Delta Practice` (with Delta bold)

---

##  Learning Outcomes

After completing this practice, you will understand:

- document.createElement()
- appendChild()
- setAttribute()
- querySelector()
- innerText vs innerHTML
- Styling elements using .style
